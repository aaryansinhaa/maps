# Assets

This folder contains various assets used by the UI.

## HTML Styles

`styles.css` contains styles which are used by the various HTML pages generated by the Python part to remove ambiguities in search results for instance.

## Map Styles

`kiwix-light` and `kiwix-dark` styles are the only two styles supported by the maps UI.

UI automatically switches between them based on UA settings.

`kiwix-light` is a direct fork of OpenFreeMap liberty theme, as maintained on https://github.com/hyperknot/openfreemap-styles/blob/main/styles/bright/README.md ; current version comes from Jan 2 2024, commit ID 851fb2e441c3aba5e3e14af13a7219124acecfc6.

`kiwix-dark` has been shamelessly created with AI and careful review with Maputnik.

### Maputnik

If you wanna test / enhance a style, you can use Maputnik online at https://maplibre.org/maputnik/

Once you've loaded the style, you just need to alter datasources to use online openfreemap sources:
- https://tiles.openfreemap.org/natural_earth/ne2sr/{z}/{x}/{y}.png for `ne2_shaded` source (natural earth tiles of the planet)
- https://tiles.openfreemap.org/planet for `openmaptiles` source (all vector tiles of the planet)
